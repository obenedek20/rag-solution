# Rag solution

To get started:
    1. Activate/create virtual env:
        - `python3 -m venv .venv`
        - `source .venv/bin/activate`
    2. Install requirements.txt: `pip3 install -r requirements.txt`
    3. Start ollama and load in ollama model:
        - `ollama serve`
        - `ollama pull qwen3:8b` or `ollama pull llama3.1:8b`
